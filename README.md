# Gwen Atelier

Статичный эскиз персонального сайта в стилистике Gwen из League of Legends.

## Что внутри

- Главная страница с приветствием, блоками «обо мне», «мои проекты», «мои ссылки» и комментариями.
- Отдельная страница постов.
- Фиксированный мини-плеер внизу страницы.
- Декоративные изображения Gwen из Riot Data Dragon.

## Запуск локально

```bash
python3 -m http.server 4173
```

После запуска откройте `http://localhost:4173`.
